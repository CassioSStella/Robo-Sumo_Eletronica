markdown# 🤖 PatoBots — Eletrônica do Robô de Sumô

Repositório da seção de eletrônica para o desenvolvimento de um robô de sumô híbrido de 500g,
desenvolvido como atividade de trainee da equipe **PatoBots**.

---

## 📋 Sobre o Projeto

O projeto consiste no desenvolvimento do hardware eletrônico do robô, incluindo esquemáticos
e layout de PCB produzidos no **KiCad**. A especificação completa do projeto está disponível
na documentação da equipe no Notion:

🔗 [Projeto de Integração Trainees PatoBots — Sumô Híbrido 500g](https://www.notion.so/Projeto-de-Integra-o-Trainees-PatoBots-Desenvolvimento-de-Sum-H-brido-500g-327fba90dc5980c68d72c41331cd149f)
